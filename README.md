# BurgerHub

A modern, scalable Flask web application for a fast food brand.

## Features

- Professional Flask app structure
- Blueprint-based routing
- Reusable Jinja templates
- Modern responsive UI
- Menu and product detail pages
- Cart and checkout placeholders

## Tech Stack

- Python
- Flask
- Jinja2
- HTML
- CSS
- JavaScript
